Это домашняя страница проекта «196 Palindrome Quest», сокращённо P196. Главная цель проекта состоит в том, чтобы проверить, сможет ли число 196 когда-либо стать палиндромом в результате многократного выполнения над ним операции RAA (Перевернуть-И-Сложить).

Также в рамках проекта проверяются другие базовые числа Лишрел: 879, 1,997, 7,059, 9,999, 10,553, 10,563, 10,577 и не только. Немного подробнее о проекте можно прочитать в блоге.

Сейчас данная страница находится в разработке и пока на ней отсутствует многое из того, что запланировано. По мере развития проекта я буду добавлять информацию. В данный момент здесь доступна только статистика по проверяемым числам, которая обычно обновляется один раз в неделю.

Исходный код проекта в репозитории iLWN пока недоступен. Файлы результатов также пока недоступны для скачивания. Среди всех натуральных чисел до 12-значных включительно проектом ALSN были обнаружены 786,640 базовых чисел Лишрел. Для каждого из них предварительно было выполнено не менее 15,000 итераций (ни одно из них при этом не стало палиндромом).

Число 196 (самое первое число Лишрел) непрерывно проверяется программой (работает 1 поток), начиная с 19 сентября 2021 года. На проверку других базовых чисел выделено меньше ресурсов. В ближайших планах выполнить по 10 миллионов итераций для всех 5- и 6-значных чисел.
