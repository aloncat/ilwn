Используемое в проекте оборудование

В настоящий момент все версии программ проектов MDPN и P196 (а также нового проекта по поиску всех базовых чисел Лишрел ALSN) работают локально. Какие-то из них многопоточные (т. е. могут задействовать для работы все или несколько ядер центрального процессора), другие — работают только в 1 потоке. Никакие из программ в данный момент не поддерживают распределённые вычисления. Также, в данный момент не используются вычисления на графических ускорителях (CUDA, OpenCL).

Ниже перечислено оборудование, которое используется проектом для вычислений или иной работы. Какая-то часть оборудования используется исключительно для нужд проекта, работая в режиме 24/7. Другая часть используется частично, лишь в то время, когда устройства включены и не используются для иных целей.

Intel Core i5-10400F / 40 Gb DDR4-2667 // 6 CPU ядер
Режим работы: 24/7. Мой основной ПК, который я использую для работы и других повседневных задач. На нём я пишу код, брожу по интернету и иногда играю в игры. Задачи проекта iLWN на нём работают в фоне практически без перерывов. [Показать подробности]
Подробная конфигурация:
Chipset: Intel B460 (Comet Lake)
RAM: 40Gb, DDR4 PC4-21300, 2667MHz, dual channel mode, timings: 16-18-18-39
CPU: Intel Core i5-10400F @ 2.90GHz / 6 cores, 12 threads / 12Mb L3 Cache / TDP 65W
GPU: GeForce GTX 1060 3Gb / GP106-300 (Pascal) / GDDR5 / 192 bits / 1152 CUDA cores
OS: Windows 10 Professional, 64-bit
Самый новый и самый производительный из всех компьютеров, участвующих в проекте. Был собран в декабре 2020 года. Его процессор Intel Core i5 относится к 10-му поколению (Comet Lake, Q2'20) и основан на усовершенствованной архитектуре Skylake. Типичная частота при полной нагрузке (12 потоков) равна 4.0GHz (максимальная частота Turbo Boost 4.3GHz).
Intel Core i5-8500B / 8 Gb DDR4-2667 // 3 CPU ядра
Режим работы: 24/7. Mac-mini с установленной на нём ОС Windows 10. Сейчас я практически не использую MacOS, поэтому этот компьютер занимается только задачами проекта iLWN. [Показать подробности]
Подробная конфигурация:
Chipset: Intel CM246 (Coffee Lake)
RAM: 8Gb, DDR4 PC4-21300, 2667MHz, dual channel mode, timings: 19-19-19-43
CPU: Intel Core i5-8500B @ 3.00GHz / 6 cores, 6 threads / 9Mb L3 Cache / TDP 65W
GPU: Intel UHD Graphics 630 (integrated)
OS: Windows 10 Home, 64-bit
Mac-mini, модель 2018 года, с процессором Intel Core i5 8-го поколения (Coffee Lake, Q2'18), основанном на архитектуре Skylake. Из-за особенностей системы охлаждения при полной нагрузке на 6 ядер издаёт очень громкий шум. Поэтому задачи выполняются не более, чем в 3 потока. При такой нагрузке типичная частота равна 4.0GHz (максимальная частота Turbo Boost 4.1GHz).
Intel Core i3-4350 / 8 Gb DDR3-1333 // 2 CPU ядра
Режим работы: 24/7. Этот компьютер занимается исключительно задачами проекта iLWN. В силу аппаратных ограничений на нём выполняется лишь часть из доступных задач проекта. [Показать подробности]
Подробная конфигурация:
Chipset: Intel H81 (Haswell)
RAM: 8Gb, DDR3 PC3-10600, 1333MHz, dual channel mode, timings: 9-9-9-24
CPU: Intel Core i3-4350 @ 3.60GHz / 2 cores, 4 threads / 4Mb L3 Cache / TDP 54W
GPU: Intel HD Graphics 4600 (integrated)
OS: Windows 10 Home, 64-bit
Старый резервный ПК, собранный ещё в 2015 году, самый слабый из всех. Его процессор относится к 4-му поколению Intel Core (Haswell, Q2'14) и основан на одноимённой архитектуре. Имеет на борту 8 Гб медленной оперативной памяти DDR3, из-за чего намного менее эффективен в некоторых задачах, требовательных к пропускной способности подсистемы памяти.
Intel Core i7-2600K / 16 Gb DDR3-1600 // 4 CPU ядра
Режим работы: частично. Второй ПК в доме. Используется другими членами семьи, часто для видеоигр, поэтому задачи проекта iLWN выполняются на нём всего по несколько часов, преимущественно в рабочие дни (менее 15% времени). [Показать подробности]
Подробная конфигурация:
Chipset: Intel Z68 (Sandy Bridge)
RAM: 16Gb, DDR3 PC3-12800, 1600MHz, dual channel mode, timings: 9-10-9-26
CPU: Intel Core i7-2600K @ 3.40GHz / 4 cores, 8 threads / 8Mb L3 Cache / TDP 95W
GPU: GeForce GTX 1060 3Gb / GP106-300 (Pascal) / GDDR5 / 192 bits / 1152 CUDA cores
GPU: Intel HD Graphics 3000 (integrated)
OS: Windows 10 Home, 64-bit
Мой старый основной ПК. Был собран осенью 2011 года. Его процессор относится ко второму поколению Intel Core (Sandy Bridge, Q1'11) и основан на одноимённой архитектуре. Имеет на борту 16 Гб оперативной памяти DDR3. Несмотря на 4 полноценных ядра и наличие технологии HT, значительно менее эффективен в задачах, требовательных к пропускной способности памяти. Типичная частота при полной нагрузке (8 потоков) составляет в среднем 3.47GHz (максимальная частота Turbo Boost 3.8GHz).
