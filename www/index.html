Блог, личная страница и проект iLWN

Большая часть содержимого этого веб-сайта приходится на мой блог, в котором я рассказываю, в основном, о моих проектах и событиях в их жизни. Помимо этого, у каждого проекта есть свой собственный раздел на сайте, где собрана вся основная информация, таблицы, графики, статистика, ссылки и т. д.

Отдельно хочется сказать о проекте iLWN. Этот акроним расшифровывается как In Love With Numbers, что в переводе на русский значит «влюблённый в числа». Это название объединяет все мои проекты, поэтому этот веб-сайт вполне можно называть домашней страницей iLWN.

iLWN — это несколько околонаучных математических проектов, построенных на общей базе исходного кода. Каждый проект занимается поиском необычных чисел. Ниже Вы найдёте подробное описание каждого проекта.

Проект MDPN – Most Delayed Palindromic Number

Поиск отложенных палиндромов, то есть таких натуральных чисел, которые становятся палиндромами после выполнения над ними N операций Перевернуть-И-Сложить (Reverse-And-Add). Главная задача проекта состоит в том, чтобы найти наименьшее число (отложенный палиндром) для каждого значения N. Вторая задача — найти некоторое количество первых отложенных палиндромов для каждого значения N.

Проект P196 – 196 Palindrome Quest

Проект P196 проверяет, станет ли число 196 (самое первое число Лишрел), а также некоторые другие числа когда-либо палиндромами в результате многократного применения к ним операции Перевернуть-И-Сложить (RAA, Reverse-And-Add).
