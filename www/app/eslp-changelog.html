ESLP software version history

ESLP is a program for finding solutions to Diophantine equations of the form p.1.n. Searches for the first (not more than 100K) primitive solutions, starting from the optionally specified value of the leading coefficient. Only 64-bit OS Windows is supported. More details at project homepage.

• • •

Version 0.20 (the latest), released on February 16, 2022.
Download this version (zip archive, 214 Kb).

Changes:

  • Use the +/- keys to change active threads.

Bug fixes:

  • Optimized memory consumption.

Features:

  • Multi-threaded computation.
  • Support for 128-bit arithmetic.
  • Max. equation degree increased to 20.
  • Max. number of coefficients is now 160.
  • Generates at least 100K solutions.

Known issues:

  • No more than 1 coefficient on the left side.
  • Specialized (optimal) algorithms are not supported.

Version 0.10, released on January 29, 2022.
Download this version (zip archive, 102 Kb).

Features:

  • First release.

Known issues:
