ESLP software version history

ESLP is a program for finding solutions to Diophantine equations of the form p.1.n. Searches for the first 100 solutions, starting from the optionally specified value of the leading coefficient. Only 64-bit OS Windows is supported. More details at project homepage.

Known issues:

  • Equation degree is limited to 9.
  • Number of coefficients is limited to 50.
  • 64-bit arithmetic is used (low values).
  • Utilizes only 1 CPU thread (slow).
